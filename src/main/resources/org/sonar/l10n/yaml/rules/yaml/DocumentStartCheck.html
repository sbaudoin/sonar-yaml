Use this rule to require or forbid the use of document start marker (---).

Parameters

present
Set to true when the document end marker is required, or to false when it is forbidden.

Examples

With present = true the following code snippet would PASS:

    ---
    this:
      is: [a, document]
    ...
    ---
    - this
    - is: another one
    ...

the following code snippet would FAIL:

    this:
      is: [a, document]
    ---
    - this
    - is: another one

With present = false the following code snippet would PASS:

    this:
      is: [a, document]
    ...

the following code snippet would FAIL:

    ---
    this:
      is: [a, document]
    ...
